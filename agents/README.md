# Agent Framework Overview

There are four provided agent archetypes, each tailored to specialized mesh roles:

### Archetypes:
1. **Alpha**: Coordinator for task routing and queue management.
2. **Beta**: GPU compute specialist (for local tasks or inference).
3. **Gamma**: Focuses on reconnaissance (OSINT).
4. **Delta**: Batch processing and data archival agent.

To define a new agent, refer to `docs/adding-an-agent.md`.
